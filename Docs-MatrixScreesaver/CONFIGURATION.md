# Configuration JSON

## `background`

| Propriété | Description |
|---|---|
| `mode` | `Single` ou `Slideshow` |
| `images` | Liste des images à afficher |
| `changeIntervalSeconds` | Durée avant changement de photo |
| `opacity` | Opacité de la photo |
| `stretch` | Cible WPF prévue : `UniformToFill` |

## `matrix`

| Propriété | Description |
|---|---|
| `glyphColor` | Couleur principale des caractères |
| `headColor` | Couleur du caractère de tête |
| `fontSize` | Taille des glyphes |
| `speed` | Vitesse globale |
| `density` | Densité des colonnes |
| `alphabet` | `Katakana`, `Hex`, `Binary`, `Runes`, `Demon` |

## `effects`

| Propriété | Description |
|---|---|
| `glow` | Halo lumineux |
| `scanLines` | Lignes CRT |
| `crt` | Prévu pour shader CRT |
| `vignette` | Assombrit les bords |
| `smoke` / `particles` | Flags de roadmap |
